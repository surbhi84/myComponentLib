Avatar

Avatars are an inseparable part of a social website, you see them here and there every day at so many platforms and obviously you need to use them as well when you are developing websites and that is why HalloUi provides you a range of avatarsto choose from and various sizes as well.

Avatar with curved edges

Simply put these are the avatars that has curved edges just add the curved-avatar class and choose a size of your choice to suit the websites's needs.

Avatar in circles

Circular avatars can be used for profile information of a user on a social web app, similar to curved-edge you can use this class too.

Avatar curved edges with badge

Badges are used to display information to the user in a sleek manner, badged convey important information to the user in a blink of an eye. Here we have two types of badges one that displays numerical information and the other displays information using the colors.
