Alert

HalloUi presents you the alert component, if there is any urgent message you need to convey to the user of your website you can use this alert. The alert component has its base paddings and margin's set so they're good to use wihtout any further filteration. You can add this class to any element of your choice, here div is used for example as div doesn't have any spacing properties addded to it beforehand.

Simple alert

Simple alerts can be used to convey quick information that can be summarised in a sentence to the user.Here you can use a font Awesome(fa) icon or google icon as well, in the example below a downloaded google icon has been used.Different colors represent different emotions and meanings so be careful in choosing which alert to use when.You can change color classes in these alerts as well, color-gp stands for green-pastel and so we have four base pastel colors, rp stands for red pastel and so on.

Detailed alert

Detailed alerts are useful to pass on a brief message to the user, you can let them know the important details in this manner along with a heading for the alert. In this component one icon svg has been used, you can choose to use an fa or google icon as well just put it in the smae line as the img tag and you're done.For the cancel icon only fa has been used.
